# To-Do List Web App

A dynamic, interactive to-do list application built with vanilla JavaScript, HTML, and CSS.

## Features

- ✅ Add new tasks
- ✅ Mark tasks as complete/incomplete
- ✅ Delete individual tasks
- ✅ Clear all tasks at once (with confirmation)
- ✅ Task counter (total and completed)
- ✅ Local storage persistence (tasks saved between sessions)
- ✅ Smooth animations and modern UI
- ✅ Light theme with purple accents
- ✅ Responsive design

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
- localStorage API
